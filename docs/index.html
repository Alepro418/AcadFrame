📌 Introducción

AcadFrame es un framework CSS/JS ligero y moderno, diseñado específicamente para sistemas académicos (CRUDs, reportes, dashboards). Ofrece componentes reutilizables, temas personalizables, modo oscuro y utilidades prácticas.

🎉 Hecho por y para estudiantes
Basado en la experiencia real de proyectos universitarios.

📦 Instalación

Copia los archivos acadframe.css y acadframe.js en tu proyecto, o usa los enlaces directos:

<link rel="stylesheet" href="acadframe/css/acadframe.css">
<script src="acadframe/js/acadframe.js" defer></script>

Opcional: importa la fuente Inter desde Google Fonts.

🎨 Variables CSS (personalización)

Puedes sobreescribir las variables en :root para cambiar colores, bordes, sombras, etc. Ejemplo:

:root {
    --ac-primary-500: #8b5cf6;
    --ac-radius-md: 0.75rem;
}

🎭 Temas y modo oscuro

Añade clase .ac-theme-emerald o .ac-theme-rose al <body>. Para modo oscuro: class="dark" o usa el botón con data-ac-darkmode-toggle.

🍞 Toast (notificaciones)

window.AcadFrame.toast('Mensaje', 'success'); // tipos: success, error, warning, info

⏳ Loader

window.AcadFrame.loader.show('Cargando...');
// ... operación asíncrona ...
window.AcadFrame.loader.hide();

📎 Exportar tabla a CSV

const tabla = document.querySelector('.ac-table');
window.AcadFrame.export.toCSV(tabla, 'reporte.csv');

✅ Validación automática

Los formularios con clase .ac-form se validan automáticamente (campos required, email, number).
